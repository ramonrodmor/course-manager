# Course manager
Arquivos do projeto do curso Introdução ao Angular 8, do Bootcamp Fullstack da DIO.

### Projeto
O projeto consiste em praticar os aprendizados do curso, construindo uma aplicação web com Angular 8 que gerencia uma lista de cursos com algumas operações.

### Tecnologias e conceitos utilizados
* Angular 8;
* Componentes;
* Segregação em módulos;
* Injeção de dependências;
* Requisições HTTP;
* Pipes;
* Rotas;
* Data binding;
* TypeScript;
* JavaScript;
* HTML5;
* CSS3.
